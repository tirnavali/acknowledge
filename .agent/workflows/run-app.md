# Running the Application

Before running the application, always ensure that the virtual environment is active.

## Steps
// turbo
1. Activate the virtual environment:
```bash
source .venv/bin/activate
```
2. Verify activation (optional but recommended):
```bash
which python
```
It should point to a path inside `.venv/bin/`.

3. Run the application:
```bash
python app.py
```
